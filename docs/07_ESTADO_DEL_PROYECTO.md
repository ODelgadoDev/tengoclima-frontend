# Estado del proyecto

## Versión funcional local

Los procesos principales están implementados y conectados a datos reales:

- Autenticación.
- Clientes.
- Cotizaciones.
- Proyectos.
- Evidencias.
- Cobranza.
- Contabilidad.
- Dashboard.
- Roles.
- Eliminación lógica.

## Estado de cierre

La etapa funcional local se considera terminada después de completar la prueba integral y corregir cualquier incidencia encontrada.

## Pendiente

Queda fuera de este cierre:

- Infraestructura de producción.
- Dominio y HTTPS.
- Base de datos definitiva.
- Servidor WSGI o ASGI.
- Almacenamiento persistente.
- Política de respaldos.
- Monitoreo.
- Configuración final de seguridad.

Estas actividades forman la siguiente etapa del proyecto.
